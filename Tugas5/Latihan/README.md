## Latihan 1 || Navigasi Dasar
`Latihan1.kt` ini mengintruksikan untuk membuat navigasi sederhana seperti  
Home -> Detail -> Back  
- Menggunakan `navigate()` untuk berpindah ke halaman detail.  
- Menggunakan `popBackStack()` untuk kembali ke halaman sebelumnya.  

## Latihan 2 || Navigasi dan Argument
`Latihan2.kt` ini mengintruksikan untuk membuat note list dan detail note.  
Note List -> Detail Note
- Menggunakan `navArgument` untuk menerima argumen dari halaman sebelumnya.  
- Menggunakan `NavType.IntType` untuk menentukan tipe data argumen.  

## Latihan 3 || Bottom Navigation
`Latihan3.kt`ini mengintruksikan untuk membuat navigasi dengan 3 bottom navigation.  
- Menggunakan `NavigationBar` untuk membuat navigasi berorientasi vertikal.  
- Menggunakan `NavigationBarItem` untuk menambahkan item navigasi ke navigasi bar.  
- Menggunakan `NavHost` untuk menentukan konten yang ditampilkan berdasarkan navigasi yang dipilih.  

## Gabungan Latihan
Terdapat 1 file `Latihan.kt` yang mengabungkan 3 komponenen tersebut agar dapat berjalan
bersama dengan benar.
